SIT 313 - Developing Client-Server Web Applications

Year: 202 Unit Information
Enrollment Modes: Trimester 2: Burwood(Melbourne), Cloud (Online)
Credit Points 1

Content

Web technologies are used to underpin many modern software applications, and understanding how to build client and server side programs is an integral part of building these applications. This unit is part of the application development major, and explores the development of web applications using the latest standards.
